## 🎲 Task 3: Order & Chaos
An asymmetric variant of Tic-Tac-Toe played on a 6x6 grid.

**Overview:**
"Order" attempts to get 5-in-a-row of either X or O. "Chaos" attempts to fill the board and prevent this. Both players have access to both pieces.

**Key Features:**
* **Two-Round System:** Play as Order in Round 1 to set a benchmark, then defend as Chaos in Round 2 against the engine.
* **Intelligent Evaluation:** Custom pattern scanning to detect straight 4s, winning 5s, and calculate round statistics.
* **Clean Architecture:** Strict separation between the React UI components and the underlying game theory logic.
